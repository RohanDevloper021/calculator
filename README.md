## 💡 About the Project

This isn’t just a calculator — it’s designed to feel alive.

From smooth animations to responsive feedback, every detail is crafted to make even basic calculations enjoyable. The goal was simple: take something ordinary and make it visually engaging, intuitive, and fun to use.

---

## 🔥 Features

- ⚡ Smooth iPhone-like animations  
- 🌌 Interactive particle background with connections  
- 🖱️ Cursor interaction effects  
- 🔊 Satisfying click & result sounds  
- ⚠️ Full shake animation on error  
- 🔴 Red glow + auto-clear error system  
- 🧠 Smart input handling (real calculator behavior)  
- 📱 Responsive & clean UI
